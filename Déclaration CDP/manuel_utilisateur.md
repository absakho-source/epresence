## 1. Introduction

### 1.1 Qu'est-ce que e-Présence ?

e-Présence est une plateforme web qui permet de gérer les feuilles de présence de manière électronique. Au lieu d'imprimer des feuilles papier et de faire signer les participants à la main, vous pouvez :

- Créer une feuille d'émargement en ligne
- Générer un QR code que les participants scannent avec leur téléphone
- Collecter les signatures électroniques en temps réel
- Exporter le tout en PDF, Excel ou JSON

### 1.2 Avantages

- **Gain de temps** : Plus de saisie manuelle
- **Écologique** : Réduction du papier
- **Traçabilité** : Horodatage précis de chaque signature
- **Accessibilité** : Fonctionne sur tout appareil connecté
- **Sécurité** : Données protégées et archivées

## 2. Premiers Pas

### 2.1 Accéder à la Plateforme

Ouvrez votre navigateur web et accédez à l'adresse de la plateforme fournie par votre administrateur.

### 2.2 Créer un Compte

1. Cliquez sur **"S'inscrire"**
2. Remplissez le formulaire :
   - Prénom et Nom
   - Adresse email professionnelle
   - Mot de passe (minimum 8 caractères)
   - Structure d'appartenance
3. Cliquez sur **"Créer mon compte"**

### 2.3 Se Connecter

1. Cliquez sur **"Se connecter"**
2. Entrez votre email et mot de passe
3. Cliquez sur **"Connexion"**

## 3. Guide Organisateur

### 3.1 Tableau de Bord

Après connexion, vous accédez à votre tableau de bord qui affiche :
- Vos feuilles d'émargement récentes
- Les statistiques de signatures
- Les boutons d'action rapide

### 3.2 Créer une Feuille d'Émargement

1. Cliquez sur **"Nouvelle feuille"**
2. Remplissez les informations :
   - **Titre** : Nom de la réunion/événement
   - **Date** : Date de l'événement
   - **Heure** : Heure de début (optionnel)
   - **Lieu** : Adresse ou salle (optionnel)
   - **Description** : Détails supplémentaires (optionnel)
3. Cliquez sur **"Créer la feuille"**

### 3.3 Partager le QR Code

Une fois la feuille créée, vous pouvez :

**Option 1 : Afficher le QR Code**
- Cliquez sur **"QR Code"** dans la liste des feuilles
- Affichez-le sur un écran ou vidéoprojecteur
- Les participants scannent avec leur téléphone

**Option 2 : Imprimer le QR Code**
- Cliquez sur **"Imprimer QR"**
- Un document A4 s'ouvre avec le QR code et les instructions
- Imprimez et affichez dans la salle

**Option 3 : Copier le Lien**
- Cliquez sur **"Copier le lien"**
- Partagez par email, SMS ou messagerie

### 3.4 Suivre les Signatures

1. Cliquez sur **"Voir"** pour ouvrir les détails de la feuille
2. La liste des signatures s'affiche en temps réel
3. Vous voyez : Nom, Prénom, Email, Téléphone, Structure, Fonction, Heure de signature

### 3.5 Exporter la Feuille

Plusieurs formats disponibles :

**PDF** (recommandé pour archivage officiel)
- Cliquez sur **"Export PDF"**
- Document A4 paysage avec toutes les signatures
- Prêt à imprimer ou archiver

**Excel/CSV** (pour analyse)
- Cliquez sur **"Export Excel"**
- Tableau avec toutes les données
- Ouvrable dans Excel, LibreOffice, Google Sheets

**JSON** (pour intégration technique)
- Cliquez sur **"Export JSON"**
- Format structuré pour développeurs

## 4. Guide Participant

### 4.1 Signer une Feuille de Présence

**Méthode 1 : Scan du QR Code**
1. Ouvrez l'appareil photo de votre téléphone
2. Pointez vers le QR code
3. Cliquez sur le lien qui apparaît
4. Vous êtes redirigé vers le formulaire

**Méthode 2 : Lien Direct**
1. Cliquez sur le lien reçu par email/SMS
2. Vous êtes redirigé vers le formulaire

### 4.2 Remplir le Formulaire

Complétez les champs demandés :
- **Prénom** * (obligatoire)
- **Nom** * (obligatoire)
- **Email** * (obligatoire)
- **Téléphone principal** * (obligatoire)
- **Téléphone secondaire** (optionnel)
- **Fonction** (optionnel)
- **Structure** * (obligatoire)

### 4.3 Dessiner votre Signature

1. Dans la zone de signature (cadre blanc)
2. Dessinez votre signature avec :
   - Votre doigt (sur mobile/tablette)
   - Votre souris (sur ordinateur)
3. Si besoin, cliquez sur **"Effacer"** pour recommencer

### 4.4 Valider

1. Vérifiez vos informations
2. Cliquez sur **"Signer"**
3. Un message de confirmation s'affiche
4. Vous pouvez fermer la page

## 5. Guide Administrateur

### 5.1 Rôles et Droits

**Utilisateur simple**
- Crée et gère ses propres feuilles d'émargement
- Exporte ses feuilles en PDF, Excel ou JSON
- Voit uniquement ses propres feuilles

**Responsable de structure**
- Voit toutes les feuilles de sa Direction/Structure
- Supervise l'activité d'émargement de sa structure
- Les feuilles restent visibles même si un agent quitte la structure

**Administrateur global**
- Accès complet à toutes les feuilles du Ministère
- Validation des nouvelles inscriptions
- Gestion de tous les utilisateurs (activer, suspendre)
- Attribution des droits de Responsable de structure

### 5.2 Gestion des Utilisateurs

1. Accédez à **Administration > Utilisateurs**
2. Vous pouvez :
   - Voir la liste des utilisateurs
   - Modifier les droits
   - Activer/désactiver des comptes

### 5.3 Paramètres

Accédez à **Administration > Paramètres** pour :
- Configurer les structures
- Modifier les textes de la plateforme
- Gérer les emails de notification

## 6. FAQ

### Questions Fréquentes

**Q : Puis-je modifier une feuille après création ?**
R : Oui, vous pouvez modifier le titre, la date, l'heure, le lieu et la description tant que la feuille est active.

**Q : Un participant peut-il signer plusieurs fois ?**
R : Par défaut, un email ne peut signer qu'une fois par feuille. Cela évite les doublons.

**Q : Comment supprimer une signature ?**
R : Seul l'organisateur ou un administrateur peut supprimer une signature depuis la vue détaillée.

**Q : Les données sont-elles sécurisées ?**
R : Oui, toutes les données sont chiffrées et stockées de manière sécurisée. L'accès est protégé par authentification.

**Q : Puis-je utiliser la plateforme hors connexion ?**
R : Non, une connexion internet est nécessaire pour accéder à la plateforme et signer.

**Q : Comment récupérer mon mot de passe ?**
R : Cliquez sur "Mot de passe oublié" sur la page de connexion et suivez les instructions.

**Q : Que se passe-t-il si un agent quitte la structure ?**
R : Les feuilles d'émargement créées restent rattachées à la structure d'origine. Le Responsable de structure continue de les voir et peut les gérer. L'agent garde accès à ses propres feuilles même après changement de structure.

**Q : Comment devenir Responsable de structure ?**
R : L'Administrateur global peut attribuer ce droit depuis la page Administration. Contactez votre administrateur pour en faire la demande.
