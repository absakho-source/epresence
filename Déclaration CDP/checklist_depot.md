## Conseils

### Pour les captures d'écran
- Utiliser des données fictives mais réalistes
- Masquer les vraies adresses email si nécessaire
- Capturer en haute résolution
- Inclure la barre d'adresse du navigateur (preuve URL)

### Pour le code source
- Exclure les fichiers sensibles (.env, config avec mots de passe)
- Inclure un fichier README.txt avec instructions d'installation
- Dater l'archive (ex: e-presence_v1.0_2026-02-12.zip)

### Pour le dépôt
- Appeler le BSDA avant pour confirmer les pièces requises
- Prévoir 1-2 heures pour le dépôt
- Garder une copie de tous les documents déposés
